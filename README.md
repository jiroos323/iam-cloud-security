# Identity & Access Management (IAM) in Cloud Security

> Learning notes and resources for a Cybersecurity Bachelor student.

This repository covers Identity & Access Management (IAM) concepts with a focus on cloud environments — particularly Microsoft Azure, but also AWS and GCP.

---

## 📂 Structure

```
├── notes/          # Theoretical notes on IAM concepts
├── labs/           # Practical exercises and walkthroughs
├── cheatsheets/    # Quick-reference guides
└── resources/      # Links, tools, frameworks and courses
```

## 🧠 Topics Covered

- Core IAM concepts (AuthN, AuthZ, Identity lifecycle)
- Azure Active Directory / Microsoft Entra ID
- Role-Based Access Control (RBAC) & Attribute-Based Access Control (ABAC)
- Zero Trust Architecture
- Conditional Access Policies
- Privileged Identity Management (PIM)
- Multi-Factor Authentication (MFA)
- OAuth 2.0, OpenID Connect, SAML
- Least Privilege Principle

Document my learning journey through cybersecurity studies, build a reference library for exams and certifications (SC-300, AZ-500), and create a portfolio that demonstrates practical security knowledge.
